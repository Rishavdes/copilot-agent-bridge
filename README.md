## 🚀 Key Features

- 🔗 **Local HTTP Bridge**  
  Connect external apps (Python AI agents) to GitHub Copilot  

- 🤖 **Multiple AI Models**  
  Supports `gpt-4o`, `gpt-4`, `o1`, `o3-mini`  

- 📊 **Real-time Dashboard**  
  Monitor requests, latency, and errors  

- 📁 **Agent Integration**  
  Manage multiple AI projects  

- 🧪 **Testing Tool**  
  Send prompts directly inside VS Code  

- 📜 **Logging & History**  
  Track requests and responses  

- ⚙️ **Custom Settings**  
  Configure port, timeout, authentication  

---

## 🛠️ How It Works

1. Start the Copilot Bridge server  
2. Send HTTP requests from your Python agent  
3. The extension routes requests to Copilot  
4. Get responses instantly
